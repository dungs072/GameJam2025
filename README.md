# GameJam2025

A Unity game project created for a game jam in 2025.

## Project Details

- **Engine**: Unity
- **Platform**: Universal Render Pipeline (URP)
- **Input System**: Unity Input System
- **Animations**: DOTween

## Setup Instructions

1. Clone this repository
2. Open the project in Unity (compatible with Unity 2022.3 LTS or later)
3. Let Unity import all packages and dependencies
4. Open the `SampleScene` in the `Assets/Scenes` folder to get started

## Project Structure

- `Assets/Scripts/` - Game scripts
  - `GamePlay/` - Core gameplay logic
  - `Constants/` - Game constants
  - `Screens/` - UI screens
- `Assets/Scenes/` - Unity scenes
- `Assets/Settings/` - Project settings and templates
- `Assets/Plugins/` - Third-party plugins (DOTween)

## Dependencies

- DOTween - Animation library
- Unity Input System
- Universal Render Pipeline (URP)
- Unity 2D packages
